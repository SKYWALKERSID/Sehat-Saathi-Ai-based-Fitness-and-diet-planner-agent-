# Sehat Saathi

AI Based Fitness & Diet Planner Agent

## Overview
Sehat Saathi is a Flask-based fitness and nutrition planner that generates personalized workout schedules, meal plans, and progress tracking views from user biometrics, goals, activity level, and dietary preferences.

## What the app does
- Builds a 3-step profile wizard for new users.
- Calculates BMI, BMR, TDEE, and calorie targets.
- Generates a 7-day workout schedule with exercise safety rules.
- Generates a 4-meal daily plan with macro targets.
- Tracks weight, water intake, sleep, and feedback.
- Supports privacy policy and terms pages inside the app.

## Design direction
- Minimal premium SaaS UI
- White cards and soft shadows
- Rounded corners and spacious layout
- Inter font
- Green accent palette
- No gradients
- No glassmorphism
- Mobile responsive layout

## Technology stack
- Backend: Python Flask
- Database: SQLite
- Frontend: HTML, CSS, Vanilla JavaScript
- Charts: Chart.js
- Layout and utilities: Bootstrap 5

## Run locally
1. Double-click `startall.bat`.
2. Wait for the local server to start.
3. Open the dashboard in the browser.
