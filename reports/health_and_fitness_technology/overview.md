# Health and Fitness Technology Overview

## Overview
Sehat Saathi is a health-tech application that blends biometric calculations, nutrition planning, workout scheduling, and progress tracking in a single product.

## Health Technology Ideas Used
- BMI, BMR, and TDEE calculations
- Personalized meal planning
- Weekly workout generation
- Daily progress logging
- Safety-aware recommendation logic

## Product Value
- Helps users plan fitness in a structured way
- Makes nutrition easier to follow
- Keeps safety considerations visible
- Improves consistency through progress tracking

## UI Direction
- Clean white-card interface
- Soft shadows and rounded corners
- Inter font
- Green accent palette
- Calm and readable dashboard layout

## Technical Base
- Flask backend
- SQLite storage
- Vanilla JavaScript interactions
- Chart.js visualizations
- Bootstrap layout support

## Conclusion
The application demonstrates a practical health-tech workflow that is easy to understand and suitable for a polished student project or startup-style demo.
